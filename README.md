# “zere，zusang和cuizi”

三者分别代指刘子越、周畅、钱子涵

释义：

## ZERE
*n.* 指特定人物，刘子越，王振宇。有时包含Mr. Soup

*adj.*

1. 指身上弥漫出特殊香气的；
2. 指做出某些沙雕行为，如“十大定律”的；
3. 指立下的目标而无法完成的；
4. 有时也用作万能形容词

**“怎么这么zere啊～”——朱明对匆匆报告有人作业不交的课代表**

## ZUSANG 
*n.* 指特定人物，周畅

*adj.* 指在上课时抄不完PPT而强行叫停老师的情况

亦有其它数词与该词读法相同，如：

轴上、周长、注上、作商

## CUIZI

*n.* 指特定人物，钱子涵
